# CIRCAI Website (Next.js App Router)

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Product deck
Place the product deck PDF at:
- `/public/decks/CIRCAI_Product.pdf`

The products page embeds this file and includes open/download links.
